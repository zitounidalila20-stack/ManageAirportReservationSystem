# ManageAirportReservationSystem
Airline Reservation System: ( NodeAir) is a modern web-based Airline Reservation System developed as a Final Year Project (PFE). The platform allows passengers to search for available flights, make reservations, complete secure online payments, and receive electronic tickets.
The system provides a user-friendly interface for passengers and a powerful administration dashboard for managing flights, reservations, users, and system operations. It also includes real-time flight status tracking, QR code ticket generation, PDF ticket export, and secure authentication using JWT.

#Features
-User registration and authentication
-Flight search and booking
-Online payment integration with Stripe
-Electronic ticket generation
-QR Code for ticket verification
-Reservation management and cancellation
-Flight status tracking
-Administrative dashboard
-Flight and reservation management
-Responsive and modern user interface
